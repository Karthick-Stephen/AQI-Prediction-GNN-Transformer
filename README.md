# AQI Prediction with GNN and Transformer

## Description
This project aims to predict Air Quality Index (AQI) using Graph Neural Networks (GNN) and Transformer models. The primary goal is to leverage the strengths of both architectures to improve accuracy and efficiency in predicting air quality levels based on historical data.

## Installation
To set up the project, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Karthick-Stephen/AQI-Prediction-GNN-Transformer.git
   ```
2. Navigate to the project directory:
   ```bash
   cd AQI-Prediction-GNN-Transformer
   ```
3. Install the necessary dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To run the model, use the following command:
```bash
python main.py
```
Ensure that you have your dataset properly formatted and available in the specified path.
